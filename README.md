# MSP Infrastructure Lab: Hybrid Identity Simulation

## Overview
A segmented virtualization environment built on Proxmox VE, utilizing pfSense for edge security and network management. This lab simulates a professional MSP environment for testing Active Directory, Entra ID integration, and Intune management.

## Project Status
- [x] Phase 1: Network Foundation & Edge Security
- [ ] Phase 2: Identity Core (Active Directory)
- [ ] Phase 3: Cloud Integration (Entra ID/Intune)

## Technical Stack
- **Hypervisor:** Proxmox VE 8.x
- **Firewall:** pfSense CE
- **Remote Access:** Tailscale (pfSense Integrated)
